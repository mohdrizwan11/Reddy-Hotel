# Reddy Hotel

Restaurant landing page for Reddy Hotel with a modern hero, menu sections, WhatsApp ordering, and contact details. Built for fast loading and mobile-first viewing.

## Features

- Hero branding with food highlights and animated collage
- Menu categories with images, prices, and WhatsApp order links
- Contact section with call and WhatsApp shortcuts
- Responsive layout optimized for mobile

## Tech Stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-ui

## Local Development

```sh
npm install
npm run dev
```

Or with Bun:

```sh
bun install
bun run dev
```

## Build

```sh
npm run build
npm run preview
```

## Deploy to Vercel

1. Import the GitHub repo in Vercel.
2. Framework preset: Vite.
3. Build command: `npm run build`.
4. Output directory: `dist`.

## Project Structure

- [src/pages/Index.tsx](src/pages/Index.tsx) – main landing page
- [src/index.css](src/index.css) – theme, animations, and base styles
- [public/favicon.ico](public/favicon.ico) – site icon
